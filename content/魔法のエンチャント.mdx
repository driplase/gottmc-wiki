このマイクラ鯖では、イベント運営によって作成された**魔法のエンチャント**がある。

## 魔法のエンチャント一覧
### 特殊エンチャント台で入手できるもの
| 名前 | 概要 | 最大レベル |
| ---- | ---- | ---- |
| [製錬](/wiki/製錬)                   | 採掘したブロックのドロップをかまどで焼いた状態にする。 | I |
| [ソニックブーム](/wiki/ソニックブーム) | 武器を使用したときにウォーデンの衝撃波を打つ。 | I |
| [精密射撃](/wiki/精密射撃)            | 矢が10秒間無重力状態になる。 | I |
| [ロングショット](/wiki/ロングショット) | 矢の初速が倍になる。 | I |
| [帰還](/wiki/帰還)                   | Mobに当たらなかった矢がテレポートし、発射時の座標に帰ってくる。 | I |
| [氷属性](/wiki/氷属性)               | 攻撃した相手を凍結させ、動きを妨ぐ。 | I |
